### 🛰️ Deskripsi Proyek

<div style="border:1px solid #ddd; padding:15px; border-radius:10px; background-color:#f9f9f9;">

Proyek ini berfokus pada **identifikasi dan pemetaan lahan sawah** dalam rangka mendukung program  
**Lahan Pertanian Pangan Berkelanjutan (LP2B)** di Kota Samarinda.

Hasil pemetaan digunakan untuk:
- 📊 Mendukung perencanaan tata ruang pertanian
- 🌾 Melindungi lahan pangan dari alih fungsi lahan
- 🗺️ Menyediakan data spasial untuk analisis GIS

</div>

---

### 🧭 Output
- Peta batas LP2B digital
- Data spasial lahan sawah
- Dokumentasi analisis GIS
